# Task 4: Classification with Logistic Regression

## Objective
Build a binary classification model using Logistic Regression on the Breast Cancer Wisconsin dataset.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Steps Performed
- Loaded and explored the dataset.
- Checked missing values.
- Removed unnecessary columns.
- Encoded the target variable.
- Split the data into training and testing sets.
- Standardized the input features.
- Trained a Logistic Regression model.
- Evaluated the model using accuracy, confusion matrix, precision, recall, and F1-score.
- Plotted the ROC curve and calculated AUC.
- Visualized the sigmoid function.

## Outcome
Successfully built and evaluated a Logistic Regression model for breast cancer classification.
